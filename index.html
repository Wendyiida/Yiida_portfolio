🔹 Project 1: Automated IP Access List Management

🛠️ Methodology & Implementation

# Reading and converting the IP list
        # Open and read allow_list.txt
        with open("allow_list.txt", "r") as file:
            ip_addresses = file.read().split()
        
        # Read remove_list.txt
        with open("remove_list.txt", "r") as file:
            remove_list = file.read().split()
        
        # Removing unauthorized IPs
        for ip in remove_list:
            if ip in ip_addresses:
                ip_addresses.remove(ip)
        
        # Convert the list back to text format
        new_content = "\n".join(ip_addresses)
        with open("allow_list.txt", "w") as file:
            file.write(new_content)

The automated system successfully filtered out unauthorized IPs and updated the access list in real-time. The system was tested with various scenarios, including:

  • Adding new IPs to the allow list
  • Removing unauthorized IPs
  • Logging access attempts
